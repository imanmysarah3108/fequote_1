Open the project folder in VS Code
Open terminal and activate virtual environment using venv\Scripts\activate
Run the backend using uvicorn api_server:app --host 0.0.0.0 --port 8000 --reload
Open browser and go to http://127.0.0.1:8000/docs to verify the API is running
Run ipconfig and check your IPv4 address
Ensure your phone and laptop are connected to the same WiFi
Update Flutter API URL using your IPv4 address (e.g. http://192.168.x.x:8000/analyze)
Open a new terminal and run the Flutter app using flutter run
Launch the app on your device
Capture an image using the camera
Wait for processing and view detected emotion with recommended quotes
